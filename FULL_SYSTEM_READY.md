## 🐛 故障排查

### WebSocket 未连接
**症状**: 显示"未连接"或灰点

**解决方案**:
1. 确认后端运行: `docker ps`
2. 检查端口 8080
3. 刷新页面重新连接
4. 查看浏览器控制台错误

### 通知不显示
**症状**: 运行脚本后没有通知

**解决方案**:
1. 确认已登录
2. 确认 WebSocket 已连接
3. 查看浏览器控制台
4. 检查通知API: http://localhost:8080/api/notifications

### 页面样式异常
**症状**: Tailwind 样式不生效

**解决方案**:
1. 重启开发服务器
2. 清空浏览器缓存
3. 检查 CSS 文件加载

### Token 过期
**症状**: 频繁跳转到登录页

**解决方案**:
1. 清空 LocalStorage
2. 重新登录
3. 检查 JWT 配置
